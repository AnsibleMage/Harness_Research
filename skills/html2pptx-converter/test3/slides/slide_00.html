화면 구성 설명

1. 코너 마커

PPTX 변환 시 좌표 매핑 정확도를 검증하기 위한 기준점. 5개 마커(TL, TR, BL, BR, CENTER)가 네 꼭짓점과 정중앙에 배치되어 변환 후 슬라이드 좌표와 1:1 매핑 여부를 확인한다.

2. 헤더 네비게이션

좌측 'TestCorp Portal' 브랜드 로고 텍스트와 우측 5개 네비게이션 메뉴(Home, Services, Board, About, Contact)로 구성. 어두운 네이비 배경 위에 밝은 텍스트 배치, 하단 파란색 구분선으로 본문과 시각적 분리.

3. 히어로 섹션

서비스 소개 및 CTA 배너. 'Welcome to TestCorp Portal' 대형 제목, 설명 부제목, 2개 CTA 버튼(Get Started 채움형, Learn More 테두리형)을 중앙 정렬로 배치. 파란 배경 위 흰 텍스트로 높은 대비 유지.

4. 통계 요약 바

핵심 성과 지표 4개(1,247 Active Users / 358 Projects / 99.9% Uptime / 24/7 Support)를 수평 균등 배치. 큰 숫자(파란, 굵은 글씨)와 작은 라벨(대문자, 회색)의 대비로 정보 위계 표현.

5. 게시판

Notice Board 제목 + '5 New' 빨간 뱃지, 5열(No, Category, Title, Date, Views) x 5행 데이터 테이블. 카테고리별 컬러 태그(Notice/Event/Update/FAQ)로 게시물 유형 시각 구분.

6. 사이드바

2개 카드: Recent Updates(최근 업데이트 4건 리스트) + Quick Links(2x2 그리드: Docs, Support, Reports, Settings). 흰 배경, 둥근 모서리, 연한 테두리의 통일된 카드 스타일.

7. 도형 & 라인 테스트

PPTX 변환 엔진의 도형/라인 렌더링 정확도 검증. 도형 5종(Rectangle, Rounded Rect, Circle, Border Only, Box Shadow) + 라인 4종(Top/Left/Bottom/Full Border). 각 요소 아래 이름 라벨 표시.

8. 푸터

4단 그리드: TestCorp 소개 / Company(About Us, Careers, Press, Blog) / Resources(Documentation, API Reference, Status Page, Changelog) / Legal(Privacy Policy, Terms of Service, Cookie Policy, GDPR). 하단 바에 저작권 + 소셜 링크(GitHub, Twitter, LinkedIn).
