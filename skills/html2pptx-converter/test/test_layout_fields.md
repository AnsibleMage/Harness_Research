# TestCorp Portal — 입력 항목 정의서

## 개요

이 문서는 TestCorp Portal 테스트 페이지의 모든 UI 요소를 정의합니다. 각 요소의 역할, 표시 형식, 데이터 내용, PPTX 변환 시 우선순위를 명시합니다.

## 슬라이드 분할 가이드

이 페이지를 PPTX로 변환할 때 권장하는 슬라이드 분할:

| 슬라이드 | 포함 영역 | 이유 |
|---------|----------|------|
| Slide 0 | 코너 마커 + 헤더 + 히어로 | 좌표 마커는 첫 슬라이드 꼭짓점/중앙에 배치 |
| Slide 1 | 통계 바 + 게시판 + 사이드바 | 콘텐츠 영역 통합 |
| Slide 2 | 도형 & 라인 테스트 | 렌더링 검증 전용 슬라이드 |
| Slide 3 | 푸터 | 다단 레이아웃 검증 |
