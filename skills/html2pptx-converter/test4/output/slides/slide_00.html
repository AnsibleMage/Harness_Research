화면 구성 설명

1. 코너 마커

슬라이드 네 꼭짓점 및 정중앙에 배치된 좌표 검증용 오버레이 마커. TL(빨강), TR(파랑), BL(초록), BR(금색), CENTER(보라) 5개 마커로 PPTX 변환 시 좌표 매핑 정확도를 검증한다.

2. 헤더 네비게이션

좌측 'TestCorp Portal' 브랜드 로고와 우측 네비게이션 메뉴(Home, Services, Board, About, Contact) 5개 링크로 구성. 어두운 네이비 배경 위에 밝은 텍스트 배치, 하단 파란색 구분선으로 본문과 시각적 분리.

3. 히어로 섹션

서비스 소개 배너. 'Welcome to TestCorp Portal' 대형 제목, html2pptx 변환 테스트 안내 부제목, CTA 버튼 2개(Get Started 채움형, Learn More 테두리형)를 파란 배경 위 중앙 정렬로 배치.

4. 통계 요약 바

핵심 서비스 지표 4개를 수평 균등 배치. 1,247 Active Users, 358 Projects, 99.9% Uptime, 24/7 Support. 큰 파란 숫자와 작은 회색 라벨의 대비로 정보 위계 표현.

5. 게시판 (Notice Board)

공지사항 테이블. 'Notice Board' 제목 + '5 New' 뱃지. 5열(No, Category, Title, Date, Views) x 5행 구성. Notice(빨강), Event(초록), Update(파랑), FAQ(노랑) 카테고리별 컬러 태그로 시각 구분.

6. 사이드바

Recent Updates 카드(최근 업데이트 4건 리스트)와 Quick Links 카드(Docs, Support, Reports, Settings 2x2 그리드) 2개 카드로 구성. 흰 배경, 둥근 모서리, 연한 테두리 카드 스타일.

7. 도형 & 라인 테스트

도형 5종(Rectangle, Rounded Rect, Circle, Border Only, Box Shadow)과 라인 4종(Top Border, Left Border, Bottom Border, Full Border)으로 PPTX 렌더링 정확도를 검증하는 전용 테스트 영역.

8. 푸터

4단 그리드 레이아웃. 컬럼1: TestCorp 소개, 컬럼2: Company(About Us, Careers, Press, Blog), 컬럼3: Resources(Documentation, API Reference, Status Page, Changelog), 컬럼4: Legal(Privacy Policy, Terms of Service, Cookie Policy, GDPR). 하단 바: 저작권 텍스트 + 소셜 링크(GitHub, Twitter, LinkedIn).
