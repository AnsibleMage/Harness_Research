# html2pptx-converter

> HTML 웹 페이지를 픽셀 수준의 정확한 레이아웃으로 PowerPoint(.pptx) 프레젠테이션으로 변환하는 Claude Code 스킬

## 개요

html2pptx-converter는 HTML 소스 파일과 보조 문서(스크린샷, 기능 설명서, 항목 정의서) 4개를 입력받아 충실한 PowerPoint 재현본을 생성합니다. **분석 → 생성 → 변환 → 수정** 4단계 Teammate 파이프라인이 전체 과정을 자율적으로 처리하며, PptxGenJS + Playwright 기반의 내장 엔진으로 외부 의존 없이 자체 완결적으로 동작합니다.

## 시작하기

### 사전 요구사항

```bash
# Node.js 16+ 의존성 설치
npm install

# Python 3.8+ 의존성 설치
pip install -r requirements.txt

# Playwright 브라우저 설치 (썸네일 생성용)
npx playwright install chromium
```

### 입력 파일 준비

폴더에 4개 파일을 준비합니다:

| # | 파일 | 용도 |
|---|------|------|
| 1 | `.html` | 변환 대상 HTML 소스 코드 |
| 2 | `.png` | HTML 렌더링 스크린샷 (검증 비교용) |
| 3 | `_description.md` | UI 각 섹션의 기능 설명서 |
| 4 | `_fields.md` | UI 요소별 역할, 입력 형식, 상호작용 정의서 |

### 사용법

Claude Code에서 스킬을 호출합니다:

```
html2pptx-converter 스킬 사용
입력 폴더: 입력파일/경로
출력 폴더: 출력파일/경로
```

4개 Phase를 자율적으로 실행하여 다음 파일을 생성합니다:

| 출력 파일 | 설명 |
|----------|------|
| `spec.json` | 입력 파일 구조화 분석 결과 |
| `slides/slide_00.html` | 엔진 호환 HTML 슬라이드 |
| `convert.js` | 자동 생성된 Node.js 변환 스크립트 |
| `presentation.pptx` | 최종 PowerPoint 파일 |
| `thumbnails.jpg` | 변환된 슬라이드 미리보기 |
| `comparison.jpg` | 원본 vs 결과 나란히 비교 |
| `validation.json` | 품질 평가 결과 (pass / needs_fix) |
