# 행안부 소프트웨어 보안약점 — CWE/OWASP/KISA 매핑

## 개요
행안부 「소프트웨어 보안약점 진단가이드」의 7유형 47개 구현단계 보안약점 전체 매핑 테이블.
Tier 1 정적 분석 스크립트(`vuln-static-scan.py`)의 검출 규칙 참조 문서.

> 최신 가이드는 구현단계 49개 항목이나, 본 스킬은 핵심 47개를 기본으로 하며 추가 2개(SSRF, 역직렬화 검증 강화)는 기존 항목에 통합 처리.
