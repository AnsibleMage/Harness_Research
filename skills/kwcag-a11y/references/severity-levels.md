# KWCAG 2.2 심각도 분류 기준

## 4단계 심각도

| 등급 | 코드 | 한국어 | WCAG 대응 | 기준 |
|------|------|--------|-----------|------|
| **Critical** | `critical` | 심각 | Level A 위반 | 콘텐츠 접근 자체가 불가능. 장애인 사용자가 해당 기능/정보를 전혀 이용할 수 없음 |
| **Major** | `major` | 주요 | Level A~AA 위반 | 접근에 상당한 어려움. 우회 방법이 있을 수 있으나 불편함이 큼 |
| **Minor** | `minor` | 경미 | Level AA 권고 | 접근에 약간의 불편. 기능 이용은 가능하나 최적의 경험이 아님 |
| **Info** | `info` | 참고 | Best Practice | 자동 판단 불가하여 수동 확인 필요. 또는 개선 권장 사항 |

## 심각도 조정 규칙

- 동일 항목이라도 **발생 빈도가 높으면** 한 단계 상향 (예: Minor → Major)
- **대체 수단이 존재**하면 한 단계 하향 가능
- **로그인/결제 등 핵심 기능**에서 발생 시 무조건 Critical
